рҹ“җ Щ…ШҙШ®ШөШ§ШӘ ЩҒЫҢШІЫҢЪ©ЫҢ
Ш§ШӘШ§ЩӮ ШЁШ§ ШіШұЩҲЫҢШі ШЁЩҮШҜШ§ШҙШӘЫҢ Ш§Ш®ШӘШөШ§ШөЫҢ
рҹ’° ЩӮЫҢЩ…ШӘ ЩҲ ШҙШұШ§ЫҢШ· Щ…Ш§Щ„ЫҢ
рҹ’Ў Ш§ШЁШӘШҜШ§ Щ…ШӘШұШ§Ъҳ ШұШ§ ШҜШұ ШЁШ®Шҙ В«Щ…ШҙШ®ШөШ§ШӘ ЩҒЫҢШІЫҢЪ©ЫҢВ» ЩҲ ШіЩҫШі ЩӮЫҢЩ…ШӘ Ъ©Щ„ ШұШ§ ЩҲШ§ШұШҜ Ъ©ЩҶЫҢШҜ ШӘШ§ ЩӮЫҢЩ…ШӘ ЩҮШұ Щ…ШӘШұ Ш®ЩҲШҜЪ©Ш§Шұ Щ…ШӯШ§ШіШЁЩҮ ШҙЩҲШҜ.

Ш¬ЩҮШӘ Ш§ШөЩ„ЫҢ ЩҲШ§ШӯШҜ (ЩҶЩ…Ш§ / ЩҫЩҶШ¬ШұЩҮ Ш§ШөЩ„ЫҢ)
Ъ©ЫҢЩҒЫҢШӘ ЩҶЩҲШұ Ш·ШЁЫҢШ№ЫҢ

ШҜШіШӘШұШіЫҢ ЩҲ Щ…ЩҲЩӮШ№ЫҢШӘ Ш§Ш·ШұШ§ЩҒ
рҹ‘Ө Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Щ…Ш§Щ„Ъ© / Щ…ШҙШ§ЩҲШұ
вҡ пёҸ Ш§ЫҢЩҶ ШҙЩ…Ш§ШұЩҮ Щ…ЩҲШЁШ§ЫҢЩ„ ЩӮШЁЩ„Ш§ЩӢ ШҜШұ Щ…Щ„Ъ©вҖҢЩҮШ§ЫҢ ШІЫҢШұ Ш«ШЁШӘ ШҙШҜЩҮ вҖ” ШЁШұШұШіЫҢ Ъ©ЩҶЫҢШҜ ШӘЪ©ШұШ§ШұЫҢ ЩҶШЁШ§ШҙШҜ:
рҹ“ё Ш№Ъ©ШівҖҢЩҮШ§ЫҢ Щ…Щ„Ъ© ЩҲ ЩҫЩҲШҙЩҮ ЩҒШ§ЫҢЩ„

ЫҢЪ© ЩҶШ§Щ… ЩҫЩҲШҙЩҮ ШЁЩҮвҖҢШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ ШЁШұШ§ЫҢ Ш§ЫҢЩҶ Щ…Щ„Ъ© ШіШ§Ш®ШӘЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ ШӘШ§ Ш№Ъ©ШівҖҢЩҮШ§ ШұШ§ ШҜШұ ШўЩҶ ЩҶЪҜЩҮ ШҜШ§ШұЫҢШҜ (Ш№Ъ©ШівҖҢЩҮШ§ ШҜШ§Ш®Щ„ ШЁШұЩҶШ§Щ…ЩҮ Ш°Ш®ЫҢШұЩҮ ЩҶЩ…ЫҢвҖҢШҙЩҲЩҶШҜ ШӘШ§ ШӯШ¬Щ… ШЁШұЩҶШ§Щ…ЩҮ ШіШЁЪ© ШЁЩ…Ш§ЩҶШҜ). ШҜШұ ШөЩҲШұШӘ ШӘЩ…Ш§ЫҢЩ„ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ЫҢЪ©ЫҢ Ш§ШІ Ш№Ъ©ШівҖҢЩҮШ§ ШұШ§ ЩҮЩ…ЫҢЩҶвҖҢШ¬Ш§ ШЁШұШ§ЫҢ ЩҫЫҢШҙвҖҢЩҶЩ…Ш§ЫҢШҙ / ШӘШ§Щ…ШЁЩҶЫҢЩ„ Ш§ЩҶШӘШ®Ш§ШЁ Ъ©ЩҶЫҢШҜ.

рҹ–јпёҸ

ШЁШұШ§ЫҢ Ш§ЩҶШӘШ®Ш§ШЁ Ш№Ъ©Ші ШӘШ§Щ…ШЁЩҶЫҢЩ„ Ъ©Щ„ЫҢЪ© Ъ©ЩҶЫҢШҜ

ЩҒЩӮШ· ШЁШұШ§ЫҢ ЩҫЫҢШҙвҖҢЩҶЩ…Ш§ЫҢШҙ ШҜШұ Ъ©Ш§ШұШӘ Щ…Щ„Ъ© Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜШӣ Ш§ШөЩ„ Ш№Ъ©ШівҖҢЩҮШ§ ШҜШұ ЩҫЩҲШҙЩҮ ЩҫЫҢШҙЩҶЩҮШ§ШҜЫҢ ЩҶЪҜЩҮвҖҢШҜШ§ШұЫҢ ШҙЩҲЩҶШҜ

рҹҸҡпёҸ

ЩҮЫҢЪҶ Щ…Щ„Ъ©ЫҢ ЫҢШ§ЩҒШӘ ЩҶШҙШҜ

ЩҒЫҢЩ„ШӘШұЩҮШ§ ШұШ§ ШӘШәЫҢЫҢШұ ШҜЩҮЫҢШҜ ЫҢШ§ Щ…Щ„Ъ© Ш¬ШҜЫҢШҜ Ш«ШЁШӘ Ъ©ЩҶЫҢШҜ

Ш¬ШІШҰЫҢШ§ШӘ Щ…Щ„Ъ©

рҹ”— Ш§ЩҒШІЩҲШҜЩҶ Щ…Щ„Ъ© Ш§ШІ Щ„ЫҢЩҶЪ© ШўЪҜЩҮЫҢ
рҹ“Ң ЩҶШӯЩҲЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ:
  1. Щ„ЫҢЩҶЪ© ШўЪҜЩҮЫҢ Щ…Щ„Ъ© ШұШ§ ШҜШұ Ъ©Ш§ШҜШұ ШІЫҢШұ ЩҲШ§ШұШҜ Ъ©ЩҶЫҢШҜ
  2. ШҜЪ©Щ…ЩҮ В«Ш§ШіШӘШ®ШұШ§Ш¬ Ш§Ш·Щ„Ш§Ш№Ш§ШӘВ» ШұШ§ ШЁШІЩҶЫҢШҜ вҖ” ЩҮЩҲШҙ Щ…ШөЩҶЩҲШ№ЫҢ Щ…ШӘЩҶ ШөЩҒШӯЩҮ ШұШ§ ШӘШӯЩ„ЫҢЩ„ Щ…ЫҢвҖҢЪ©ЩҶШҜ
  3. Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШҙЩҶШ§ШіШ§ЫҢЫҢвҖҢШҙШҜЩҮ ШұШ§ ШЁШұШұШіЫҢ Ъ©ЩҶЫҢШҜ ЩҲ ШҜШұ ШөЩҲШұШӘ ЩҶЫҢШ§ШІ ЩҲЫҢШұШ§ЫҢШҙ Ъ©ЩҶЫҢШҜ
  4. ШҜЪ©Щ…ЩҮ В«Ш§ЩҒШІЩҲШҜЩҶ ШЁЩҮ Щ„ЫҢШіШӘВ» ШЁШұШ§ЫҢ Ш°Ш®ЫҢШұЩҮвҖҢЫҢ Ш®ЩҲШҜЪ©Ш§Шұ Ш§ШіШӘ
вҡ пёҸ Ш§ЫҢЩҶ ЩӮШ§ШЁЩ„ЫҢШӘ Ш§ШІ Claude API Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢЪ©ЩҶШҜ. ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ШұШ§ЫҢЪҜШ§ЩҶШҢ Ъ©Щ„ЫҢШҜ API Ш®ЩҲШҜ ШұШ§ Ш§ШІ console.anthropic.com ШҜШұЫҢШ§ЩҒШӘ ЩҲ ЫҢЪ© ШЁШ§Шұ ШҜШұ Ъ©Ш§ШҜШұ ШІЫҢШұ ЩҲШ§ШұШҜ Ъ©ЩҶЫҢШҜ (ШЁЩҮ ШөЩҲШұШӘ Щ…ШӯЩ„ЫҢ Ш°Ш®ЫҢШұЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ Ш¬Ш§ЫҢЫҢ Ш§ШұШіШ§Щ„ ЩҶЩ…ЫҢвҖҢШҙЩҲШҜ).
вҡҷпёҸ
ШҜШұ ШӯШ§Щ„ ШҜШұЫҢШ§ЩҒШӘ ЩҲ ШӘШӯЩ„ЫҢЩ„ ШөЩҒШӯЩҮ...
вң… Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Ш§ШіШӘШ®ШұШ§Ш¬ ШҙШҜ вҖ” ШЁШұШұШіЫҢ ЩҲ ШӘШЈЫҢЫҢШҜ Ъ©ЩҶЫҢШҜ
рҹ“Ӣ JSON Ш®Ш§Щ…: Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ЫҢЩҶ JSON ШұШ§ Ъ©ЩҫЫҢ Ъ©ЩҶЫҢШҜ ЩҲ Ш§ШІ Ш·ШұЫҢЩӮ В«ШЁШ§ШұЪҜШ°Ш§ШұЫҢ ЩҫШҙШӘЫҢШЁШ§ЩҶВ» ШҜШұ ШөЩҒШӯЩҮ Щ„ЫҢШіШӘ ЩҶЫҢШІ ЩҲШ§ШұШҜ Ъ©ЩҶЫҢШҜ.
